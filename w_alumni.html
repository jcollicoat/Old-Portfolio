Chico State Alumni




< brief >

Create an interactive web data visualisation which displays information drawn from a database.


< concept >

For this project I had to create three visualisations of a large database containing information on Chico State design alumni. Required information included gender, discipline within design, and the state in which former students currently work. The software used was D3 Javascript embedded in HTML, which allowed for a smooth, responsive experience.
